**C# консольное приложение** для мониторинга землетрясений в реальном времени с использованием **паттерна Strategy (Стратегия)**.

Приложение получает данные с официального USGS API (United States Geological Survey) и анализирует их с помощью разных алгоритмов оценки опасности.

## Возможности

- Получение актуальных данных о землетрясениях со всего мира каждые 15 секунд
- Поддержка 4 разных стратегий анализа:
  - По магнитуде
  - По свежести событий
  - По кластерам (скоплениям) в регионах
  - По глубине очага
- Динамическая смена стратегии во время работы программы
- Кэширование событий (до 200 последних уникальных землетрясений)
- Уровни тревоги: `Info` → `Watch` → `Warning` → `Critical`
- Защита от падения при недоступности API (fallback-данные)

## Паттерн Strategy — как это работает

**Паттерн Strategy** позволяет менять алгоритм анализа землетрясений «на лету», не меняя код самого монитора.

### Почему используется Strategy?

Вместо того чтобы писать огромный `if-else` внутри монитора, вся логика анализа вынесена в отдельные классы-стратегии.  
Это делает код:
- Гибким
- Расширяемым
- Легко тестируемым
- Понятным

### Структура стратегий

Все стратегии реализуют интерфейс `IEarthquakeStrategy`:

```csharp
public interface IEarthquakeStrategy
{
    string Name { get; }
    AnalysisResult Evaluate(IReadOnlyList<EarthquakeEvent> events);
}
```

**Доступные стратегии:**

| Команда | Стратегия           | Что анализирует                              | Когда выдаёт Critical |
|--------|---------------------|---------------------------------------------|-----------------------|
| `1`    | **MAGNITUDE**       | Самое сильное землетрясение                 | M ≥ 6.5              |
| `2`    | **RECENCY**         | Свежесть + количество событий за час        | ≥ 20 событий за час  |
| `3`    | **CLUSTER**         | Самый активный регион (кластер)             | ≥ 10 событий в регионе |
| `4`    | **DEPTH**           | Самое мелкофокусное (shallow) землетрясение | Глубина ≤ 10 км + M ≥ 5.5 |

##  Как запустить

1. Склонируй или скачай проект
2. Открой решение в **Visual Studio** или используй .NET CLI
3. Запусти проект (`EarthquakeIntelligenceMonitor`)

```bash
dotnet run
```

## 📖 Использование

После запуска вы увидите заголовок и список команд:

```
Commands:
1 - Magnitude strategy
2 - Recency strategy
3 - Cluster strategy
4 - Depth strategy
q - quit
```

### Управление во время работы:

- Нажми **`1`** → переключиться на анализ по магнитуде (по умолчанию)
- Нажми **`2`** → переключиться на анализ по свежести
- Нажми **`3`** → переключиться на поиск кластеров
- Нажми **`4`** → переключиться на анализ по глубине
- Нажми **`q`** → завершить работу программы

Каждые 15 секунд приложение:
- Загружает свежие данные с USGS
- Добавляет новые события в кэш
- Анализирует данные текущей стратегией
- Выводит результат с уровнем тревоги и сводкой

## Архитектура проекта

- **`EarthquakeEvent`** — модель данных одного землетрясения
- **`IEarthquakeStrategy`** — интерфейс всех стратегий
- **`EarthquakeContext`** — контекст, который хранит текущую стратегию
- **`MagnitudeStrategy`**, **`RecencyStrategy`**, **`ClusterStrategy`**, **`DepthStrategy`** — конкретные алгоритмы
- **`UsgsEarthquakeApi`** — клиент для работы с USGS API
- **`EarthquakeMonitor`** — основной цикл мониторинга
- **`AppRunner`** — запуск приложения и обработка ввода пользователя
